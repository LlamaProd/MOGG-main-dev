c_variation_dairy

	!! Creature variations for consistent editing of raws, not all of these will always need to be used, but when used, best practice is to fill all arguments, and use a placeholder like "X" for any unused arguments rather than leaving them blank. !!
	
	!! The core mod is simply focussed on correcting the flat tick rate of 20000 for all milkable creatures, adding up to four preferences for modded creatures that have less than that, and fixing two (probable) bugs in the game; milkable animal men, and small animals not dropping any meat when butchered. !!
	
	!! The main dairy "expansion" for the mod aims to tweak how milking and dairy is categorised, increase value of products in some cases, and add more mammals to the list of creatures that can be milked. The more elaborate of the creature variations below are designed with that in mind !!


[OBJECT:CREATURE_VARIATION]

		!! This is the "master" variation for MOGG creature edits, it covers almost everything using arguments but there will likely still be some edge-cases where more modular variations or custom CV/APPLY_CURRENT is required or just preferred. Some of the smaller variations will often do everything you need with less arguments required, depending on the creature. Best to fill unused arguments with arbitrary character like "X" to avoid argument bugs. (The CREATURE_EDIT variation, below this one, is the go-to default variation for working with anything that has a primary, gendered and child name structure with regular "s" plurals for all names).
			!! More detailed information about the intended use of dairy products and definitions can be read in the comment above custom dairy varaiations below these. !!
			
		!! Argument Key:
			!ARG1 = what naming convention the creature conforms to:
				R = regular plurals (dog, dogs)
				I = any irregular plurals (cavy, cavies/woman, women/puppy, puppies)
				F/M suffix = if the male or female name is the same as the main animal name e.g. (IM would apply to dogs, RF would apply to cats)
			!ARG2 = the primary name (dog/cavy/seal)
			!ARG3 = primary name iregular plural (cavies). If you are not using I/IM/IF in !ARG1 just fill with abitrary character like "X"
			!ARG4/5 = secondary female name & irregular plural (woman, women) as above "X" will suffice for !ARG5 using regular plurals.
			!ARG6/7 = same as above but for males
			!ARG8/9 = same as above but for child names (puppy, puppies/calf, calves)
			!ARG10 = only triggers if !ARG12=M , the creature class tag, (HERBIVORE, CARNIVORE_EXOTIC, OMNIVORE_LUXURY) this is important if adding milk as the CREAURE_CLASS here defines regular, exotic, luxury depending on suffix). If referencing class for a new milk, it will need to be added to creature_dairy_generic.
			!ARG11 = should pre-existing dairy materials be removed?
				DR = "Dairy remove/dairy removal". Any value other than "DR" will skip this.
			!ARG12 = make the creature milkable and assign dairy materials based on !ARG10
				M = milkable, any other value will skip this.
			!ARG13 = milking tick rate. This is based on my own calculations, but for reference a cow is 20000. Leave as "X" if skipping !ARG12
			!ARG14 = if you want to alter the default name, generally for adding a collective adjective such as "feline" to a creature. Will only with this variation if !ARG2/!ARG3 match the creature in the raw you are editing. Can be skipped with "X"
				NR = regular creature plurals
				IR = irregular creature plurals
			!ARG15 = collective adjective for the creature (feline/musteline/phocine etc.)
					Be advised, you can just end your aruments early and finish with a ] but this can sometimes cause oddities to the raw edits, especially if done too early. !!

	!! These variations mostly concern the dairy expansions for MOGG !!	
		!! Broadly speaking the use of "exotic" and "luxury" classifications in the following variations are for use with GIANT creature variations and creatures too small to be milked more than once, respectively. These also refer to custom milk and cheese materials in material_template_dairy. Creatures are also given a new CREATURE_CLASS tag, generally HERBIVORE, CARNIVORE or OMNIVORE which also refers to pre-defined creature dairy materials referenced in creature_dairy_generic. These are key to keeping the plethora of dairy products within the bounds of broad classifications, and hopefully improving compatibility and moddability in the future.
		
		!! The following variations are for custom dairy material names, if you don't want to use generic milk materials, for whatever reason; e.g. elephant so that it's 5x material value multiplier identifies it as extra-special. !!
			!! The fourth argument is where a one of the new calculated milking tick rates should be added. !!
			!! !ARG1=dairy creature class name (e.g. OMNIVORE) !ARG2 = name, !ARG3 = adjective, !ARG4 = milking tick rate, !ARG5="N"ONE, "E"xotic or "L"uxury for the prefixes, and correct template settings. Will have errors if !ARG5 isn't filled with one of the designated character arguments. !!
[CREATURE_VARIATION:CUSTOM_DAIRY]
	[CV_NEW_TAG:CREATURE_CLASS:!ARG1]
	[CV_NEW_TAG:SELECT_CASTE:FEMALE]
		[CV_NEW_CTAG:5:N:USE_MATERIAL_TEMPLATE:MILK:MILK_TEMPLATE]
		[CV_NEW_CTAG:5:E:USE_MATERIAL_TEMPLATE:MILK:MILK_E_TEMPLATE]
